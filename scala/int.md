# Int

```scala
import scala.math

println(abs(-1)) // 絶対値
println(max(1,2)) // 最大値
println(min(1,2)) // 最小値
println(pow(2, 10)) // 乗数
println(sqrt(2.0)) // 平方根

println(floor(9.9)) //=> 9.0 (切り捨て)
println(ceil(9.9)) //=> 10.0 (切り上げ)
println(round(9.9)) //=> 10 (四捨五入)

println(E) // 自然対数
println(Pi) // 円周率
println(random) //=> [0,1) の乱数
```
