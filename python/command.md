# Python Command

### 3rd party module install

`pip install [module name]`

### uninstall

`pip uninstall -y [module name]`

### module list

`pip list`

## 仮想環境

### 仮想環境作成

`python -m venv [folder name]`

### 仮想環境確認

`dir [folder name]`

### 仮想環境有効化

`[folder name]\Scripts\activate.bat`

### 無効化

`deactivate`
